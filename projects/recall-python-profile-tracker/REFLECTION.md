# Project Reflection

## Summary

This project helped turn basic Python syntax into a small, structured program. The main improvement was moving from a dictionary of separate lists into a list of dictionaries, which made each learner profile easier to manage.

## Key Takeaways

- Lists are useful for collections of related values.
- Dictionaries are useful for labeled information.
- A list of dictionaries is a strong pattern for storing multiple records.
- F-strings make output easier to read.
- `.join()` is useful when displaying list items as a single readable line.

## Improvement Notes

The first version had two issues:

- Some skills were stored as one long string instead of separate list items.
- One f-string used conflicting quote styles.

Both issues were fixed in the final version.
