# Project Reflection

## Summary

This project applied loop logic to a practical budgeting example. The final script uses a list of dictionaries to store category-level expense records, then loops through each record to calculate summary values.

## Key Takeaways

- A list of dictionaries is useful for storing structured records.
- A running total can be updated inside a loop.
- Highest and lowest records can be tracked by comparing each record against the current saved value.
- Conditional logic can turn a numeric result into a readable budget status.

## Improvement Notes

The first version mixed variable names and generator expressions in a way that made the logic harder to follow. The final version uses a direct loop, clear variable names, and explicit comparison logic.
